# Screen Translation Workspace

Turn product screenshots into organized, editable translation steps for your team.

## What it does

1. Upload a screenshot (or paste from clipboard)
2. AI reads the screen and translates visible UI text
3. Save each screen as a step in a flow
4. Teammates review, edit, and comment

- Next.js (App Router) + TypeScript + Tailwind + shadcn/ui
- Supabase (Auth, Postgres, Storage)
- Vision model for screenshot → structured translation JSON
