# Screen Translation Workspace

Turn product screenshots into organized, editable translation steps for your team.

## Docs

- **[Minimum MVP (v0)](./MINIMUM_MVP_PLAN.md)** — what to build first
- **[Full product vision](./MVP_PLAN.md)** — north star for later
- **[Build plans](./plan/README.md)** — day-by-day implementation

## Stack

- Next.js (App Router) + TypeScript + Tailwind + shadcn/ui
- Supabase (Auth, Postgres, Storage)
- Vision model for screenshot → structured translation JSON

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Status

Day 1 in progress — foundation and auth.
